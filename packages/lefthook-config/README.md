# @alex-grover/prettier-config

A [Lefthook](https://lefthook.dev) configuration for ESLint, Prettier, and Stylelint.

## Installation

```sh
bun install --dev lefthook @alex-grover/lefthook-config
```

## Usage

Add the following entry to your `lefthook.yml`:

```yaml
extends:
  - node_modules/@alex-grover/lefthook-config/config.yml
```
